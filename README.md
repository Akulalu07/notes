Конспекты 


Если видите ошибку в конспектах, то пишите на почту или открывайте Issues 
Если у вас есть недостоющие здесь конспекты, то сделайте fork и закиньте что считаете нужным, затем сделайте pull request (через 2-3 дня будет обработано и добавлено красиво)

Часть конспектов этот чисто формулы так как переписывать в электронную форму мне иногда лень.
